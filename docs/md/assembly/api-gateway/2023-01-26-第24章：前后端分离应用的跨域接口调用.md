## 一、学习指引

在使用前后端分离的方式构建运营后台应用系统以后，你会遇到一个非常常见的问题，就是**跨域访问**。那么什么是跨域访问呢？

**跨域访问**是指当一个网页从一个域名（或端口）请求另一个域名（或端口）的资源时，由于浏览器的同源策略限制，请求会被拒绝。跨域访问是一种常见的安全限制，用于防止网页在不受信任的域中访问敏感信息。

那么我们本章节就会遇到这样的一个前端应用访问后端接口所出现的跨域问题。
